# eslint-config-jet

Extend `eslint-config-jet` in your `.eslintrc` file.

# Installation

You can install [ESLint] with yarn:

```sh
yarn add --dev eslint \
eslint-config-jet \
eslint-plugin-babel \
eslint-plugin-prettier \
babel-eslint \
prettier
```

# Editor Integration

* [linter-eslint] for Atom's [Linter][atom-linter].
